DICAS DE HTML PARA QUEM ESTÁ APRENDENDO

" Olá pessoal, meu nome é Matheus Rodrigues e neste site mostrarei dicas e conceitos importantes para quem esta começando a usar HTML.Por incrível que pareça, sou um iniciante também hahah. Porém, vou passar pra vocês tudo que eu aprendi. "

Sem mais delongas, vamos às codificações básicas e seus respectivos significados.

Tags importantes e seus conceitos:

<!doctype html>
Tag mais importante, no qual ajuda o navegador a interpretar o documento como um documento html.
<html>
Tag que englobará todas as codificações que fizer pro site, indicando tudo que está dentro dela como html. Fica logo abaixo da tag "!doctype html".
<head>
Tag de cabeçalho em que dentro dela é feito declarações que normalmente não são visíveis. Nela são declaradas tags "meta" com seus atributos.
<title>
Tag do título da parte superior do site, na aba, na qual facilita o navegador de acha-lo.
<meta charset>
A tag meta representa vários tipos de metadados, ou seja dados sobre seus dados.
Ex.: palavras chave, resumo do conteúdo. Já o atributo charset(conjunto de caracteres) indica o padrão de caracteres, que no caso é o formato "utf-8", ou seja se eu tivesse feito uma pagina com caracteres de uma lígua não nativa teria que usar essa codificação.
<body>
Tag que efetivamente vai ser exibido todo o conteúdo visual do site, sendo o corpo do site.

Observações Importantes:

  1. Ao utilizar qualquer tag, deverá sempre usar ela(tag) entre o sinal de <(menor) e >(maior), exceto tags de "meta".
  2. Para o fachamento de uma TAG, deve-se usar o sinal <(maior), em seguida /(barra) depois a tag e no final o >(menor).
  3. Para TAGS self close (aberta e fechada em uma só TAG) deverá usar <(TAG) />.
    Exemplo: a TAG "br" que é usada para quebrar um parágrafo.
  4. Caso queira adicionar um comentário em seu códico, usa-se <.!-- Comentário --> sem o ponto.
    Importânte: o comentário não afetará no navegador, pois não é processado.
